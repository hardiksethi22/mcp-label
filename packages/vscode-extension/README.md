# @mcp-label/vscode-extension

VS Code extension for [mcp-label](https://github.com/mcp-label/mcp-label) — a privacy, safety, and permissions label for MCP servers.

> ⚠️ **This package is a scaffold.** The VS Code extension is not yet implemented.

## Planned features

- Inline permission labels in MCP config files
- Hover tooltips showing risk and permissions for each server
- Warnings for unpinned packages, broad filesystem mounts, and secrets
- Quick-fix suggestions for hardening
- Command palette integration for scanning and exporting reports
