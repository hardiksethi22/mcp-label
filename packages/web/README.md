# @mcp-label/web

Web app for [mcp-label](https://github.com/mcp-label/mcp-label) — a privacy, safety, and permissions label for MCP servers.

> ⚠️ **This package is a scaffold.** The web app is not yet implemented.

## Planned features

- Paste an MCP config and render a safety label in the browser (client-side only)
- Upload a JSON report and view the rendered label
- Browse schema documentation
- Generate README badge snippets
- Search public labels (future hosted registry)

## Design principles

- **Client-side first** — MCP configs are parsed in the browser, not sent to a server
- **No backend required** — works as a static site
- **Privacy-preserving** — no config data leaves the browser
