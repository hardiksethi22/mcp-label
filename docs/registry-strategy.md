# Registry Strategy

## Vision

`mcp-label` aims to become compatible with the MCP Registry for public label hosting, server card generation, and trust badges.

## Planned Features

- Generate standardized Server Cards conforming to `mcp-server-card.schema.json`
- Publish labels as static JSON files for registry indexing
- Generate embeddable README badges (SVG)
- Support registry search and comparison views

## Current Status

The registry strategy is planned for v0.6. The JSON Schema for server cards (`schemas/mcp-server-card.schema.json`) is defined.
